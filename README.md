# graphyte
TestNG (Java) Automation Framework

## Scope
```
This is to perform all manner of automation tests using TestNG
```
## Requirements
```
1. Download an IDE like Eclipse, Intelli-J, or Visual Studio Code
2. Have the lastest version of Java: check by typing java --version in console
3. Import the TestNG Libraries
4. Install the JAVA Test Runner: https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-test
5. Install language support for Java: https://marketplace.visualstudio.com/items?itemName=redhat.java
6. Add the JAR files that contain the Selenium API: https://selenium.dev/downloads/
7. Right-click on the "src" package folder then choose New > Other :: TestNG Class
8.a. Set package name to something relevant
8.b. Set filename to something appropriate, starting with "test_"
```
